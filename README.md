# 📊 Sales Performance Dashboard

An interactive **Tableau dashboard** built to analyze sales performance, covering customers, employees, suppliers, product categories, and geographic distribution of sales.

## 📌 Overview

This project transforms raw sales data into clear, actionable insights that support business decisions — such as which products perform best, which markets are growing, and how the sales team is performing.

## ✅ Key Metrics

- **Total Customers:** 91
- **Total Employees:** 9
- **Total Quantity Sold:** 51,317
- **Total Suppliers:** 29

## 🔍 Dashboard Features

- **Category Sales Distribution** — Breakdown of sales across Beverages, Dairy Products, Confections, Seafood, Grains/Cereals, Meat/Poultry, Condiments, and Produce
- **Sales by Country** — Interactive world map showing geographic sales distribution
- **Sales Trend** — Quarterly trend analysis of quantity sold over time
- **Top 5 Performance Chart** — Best-performing employees by sales quantity
- **Top 10 Selling Products** — Ranking of the most sold products
